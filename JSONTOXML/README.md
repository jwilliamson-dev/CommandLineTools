# JSON TO XML Converter
Jacob Williamson

Drexel University

Summer 2021

## NAME
    jtox - JSON to XML converter
## SYNOPSIS
    jtox [FILE]...
## DESCRIPTION
    Convert prettified JSON to XML. The top-level element is labeled
    "root" and items in an array are labeled "item"
### Exit Status:
        0 if OK,
        1 if error. 
## BUILD
    Run make. Outputs jtox in the current directory.
