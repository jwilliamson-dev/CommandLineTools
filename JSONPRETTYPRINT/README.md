## NAME
    jsonpretty - JSON Pretty Printer
## SYNOPSIS
    jsonpretty [FILE]...
## DESCRIPTION
    Prints valid JSON in pretty format.
### Exit Status:
        0 if OK,
        1 if error. 
## BUILD
    Run make. Outputs jsonpretty in the current directory.
